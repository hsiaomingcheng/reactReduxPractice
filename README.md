## 安裝redux與react-redux
    # NPM
    npm install redux
    npm install react-redux

    # Yarn
    yarn add redux
    yarn add react-redux

## 安裝Redux-Saga
    $ npm install --save redux-saga
    or
    $ yarn add redux-saga
